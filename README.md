# My dotfiles

My personal development environment setup with Zsh and Oh My Zsh.

## Dependencies
To use some resources of this project, you need do have it in your environment:
- curl (to download some resources and packages)

## Installation
Just run these commands and you're good to go
```zsh
git clone https://github.com/ViniciusCassemira/dotfiles.git
```

```zsh
cd dotfiles
```

```zsh
chmod +x setup.sh
```

```zsh
./setup.sh
```

```zsh
./setup2.sh
```

```zsh
source ~/.zshrc
```
